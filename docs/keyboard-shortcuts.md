# Keyboard Shortcuts Reference

Complete reference for all keyboard shortcuts in the Brainstorm App.

## Global Shortcuts

These shortcuts work anywhere in the application (except when editing text in an input or textarea).

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` / `Cmd+Z` | Undo | Revert the last action (up to 50 levels) |
| `Ctrl+Shift+Z` / `Cmd+Shift+Z` | Redo | Re-apply the last undone action |
| `Ctrl+Y` / `Cmd+Y` | Redo | Alternative redo shortcut |

## Canvas Shortcuts

These shortcuts work when the canvas is focused and no text input is active.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Delete` / `Backspace` | Delete selected | Delete all selected nodes or edges |
| `Enter` | Edit selected node | Start editing the single selected node |
| `Escape` | Stop editing / Deselect | Exit edit mode or clear selection |

## Node Interactions

| Interaction | Action | Description |
|-------------|--------|-------------|
| **Double-click** canvas | Add node | Create a new node at the click position |
| **Double-click** node | Edit node | Start inline text editing |
| **Double-click** edge label | Edit label | Start editing the edge label |
| **Right-click** node | Context menu | Open the node actions menu |
| **Shift+Click** node | Multi-select | Add node to selection (or remove if already selected) |

## Context Menu Navigation

When the right-click context menu is open:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Arrow Up` | Previous item | Move focus to the previous menu item |
| `Arrow Down` | Next item | Move focus to the next menu item |
| `Enter` / `Space` | Activate | Click the currently focused item |
| `Escape` | Close | Close the context menu |
| `Tab` | Next item | Move to the next item (wraps around) |
| `Shift+Tab` | Previous item | Move to the previous item (wraps around) |

## Inline Editing

When editing a node's text:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Enter` | Save and close | Save the text and stop editing |
| `Shift+Enter` | New line | Insert a newline (text area) |
| `Escape` | Cancel | Discard changes and stop editing |

## Edge Label Editing

When editing an edge label:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Enter` | Save and close | Save the label and stop editing |
| `Escape` | Cancel | Close without saving |

## Modal Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Escape` | Close modal | Close any open modal dialog |

## Shortcuts Tooltip

The keyboard shortcuts tooltip can be opened from the toolbar by clicking the keyboard icon. It displays the same shortcuts listed here for quick reference.
